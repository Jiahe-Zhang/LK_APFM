# 一、LK_APFM是什么？
LK_APFM是一个自动宠物喂食器项目，基于涂鸦三明治开发套件制作。LK是我家的狗主子“莱卡”的名字拼音首字母。
# 二、关于LK_APFM
- 对象：主子（宠物）
- 场景：家庭
- 作用：
-   定时、定量投放食物
-   通过安卓APP即时投放食物
-   语音控制即时投放食物
-   语音输出执行结果
